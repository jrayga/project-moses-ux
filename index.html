    • About

    Project Moses is a Crisis Response Toolbox that aims to be an all-in-one web dashboard for Filipinos in times of calamity and crisis mainly and should still be relevant even in times of peace. We are starting with a Virtual Triage that could match patients with hospitals. It is a publicly accessible collaboration tool using Information Communication Technology (ICT) meant to connect COVID-19 response actors.
